## 一、给 AI 的总提示词（复制整段）

```markdown
# 任务：首页仪表盘 + 数据统计页 UI/功能迁移 — 对齐 tuandui

## 背景
sibling 项目的 dashboard / statistics 常见问题是：
- 裸 QLabel 堆数字、无卡片层次、表格嵌套滚动条
- 无 WelcomeBanner、无 Loading/Error/重试
- 统计页与首页数据结构不一致、Tab 内表格高度错乱
- 未接 `/statistics/overview` 等后端接口

## 总目标
1. **首页 dashboard**：蓝渐变欢迎条 + StatCard 4×2 网格 + SectionCard 分区 + Top5 热门表
2. **统计 statistics**：PageContainer 整页滚动 + 概况/互动 StatCard + QTabWidget 热门 Top10 + 搜索词分析
3. 复用 `StatisticsService` + `StatisticsModels`，权限 `statistics:view`
4. objectName / QSS 与全站一致（WelcomeBanner、StatCard、SectionCard、DataTable）

## 第一步：复制组件与模块
- `common/widgets/StatCard.*`
- `common/widgets/PageContainer.*`
- `modules/dashboard/StatisticsService.*`
- `modules/dashboard/models/StatisticsModels.*`
- `modules/dashboard/DashboardPage.*`
- `modules/statistics/StatisticsPage.*`
- CMakeLists / PageRouter 注册 `dashboard`、`statistics`

## 第二步：按本文档 §三 线框改 layout
- dashboard → 模板 A（无页面顶栏标题，WelcomeBanner 即首屏）
- statistics → 模板 A + 页内 PageTitle + Ghost 刷新

## 第三步：接 API（§四）
- dashboard：仅 `GET /statistics/overview`（含 topHotArticles Top5）
- statistics：overview + hot-article×6 排序 + hot-keyword

## 硬规则
| 规则 | dashboard | statistics |
|---|---|---|
| 外层滚动 | PageContainer，margin 28,24,28,28 | 同左 |
| 指标 | StatCard 4 列网格，spacing 14 | 同左 |
| 分区 | SectionCard + SectionTitle | 同左 |
| 表格 | DataTable，无 grid，斑马纹 | Tab 内表格禁用纵向滚动，固定行高撑开 |
| 权限 | 无 statistics:view 仅显示 WelcomeBanner | 无权限 ErrorLabel |
| 数字 | QLocale 千分位 | 同左 |
| 按钮 | GhostButton「重新加载/刷新」 | 同左 |

## 禁止
- StatCard 内联 setStyleSheet 改颜色
- 统计 Tab 内 QTableWidget 自带纵向滚动（应整页滚动）
- dashboard 与 statistics 各写一套不同的 overview 解析
- 热门表用默认 QTableWidget 样式（无 DataTable）

## 交付
1. DashboardPage + StatisticsPage 源码
2. StatisticsService 对接完成
3. §五 验收清单全部勾选
4. 截图：有权限 / 无权限两种状态

请先对比 sibling 现有 dashboard/statistics 源码，输出差异表，再实施。
```

## 二、两页定位差异

| 维度 | dashboard（首页） | statistics（数据统计） |
|---|---|---|
| route | `dashboard` | `statistics` |
| 菜单 | 工作台 · 首页仪表盘 | 统计与开放 · 数据统计 |
| 权限 | 有 `statistics:view` 才显示数据区 | 必须 `statistics:view` |
| 数据接口 | 仅 `overview`（1 次请求） | `overview` + 6×`hot-article` + `hot-keyword`（8 次） |
| 热门榜 | Top **5**（overview 内嵌） | Top **10**，6 个 Tab 维度 |
| 搜索词 | 无 | 热门词 + 无结果词 Tab |
| 顶栏 | 无 PageTitle（靠 WelcomeBanner） | `PageTitle`「数据统计报表」+ Ghost 刷新 |
| 无权限 | 仍显示欢迎条，hint 提示无统计权限 | ErrorLabel 提示无权限 |

## 三、布局线框与 objectName

### 3.1 首页 dashboard（模板 A）

```
QVBoxLayout(0,0,0,0)
└ PageContainer → #PageContent (margin 28,24,28,28 spacing 18)
     ├ #WelcomeBanner
     │    ├ #WelcomeTitle     「你好，{realName}」
     │    └ #WelcomeHint       有数据/无权限不同文案
     ├ #LoadingLabel           加载中（默认隐藏）
     ├ #ErrorLabel + GhostButton「重新加载」
     └ statsPanel（有数据后 visible）
          ├ SectionCard「知识概况」
          │    └ QGridLayout 4列 × StatCard×8
          │         知识总数 | 已上线 | 待审核 | 草稿
          │         今日浏览 | 今日搜索 | 已下线 | 已驳回
          ├ SectionCard「互动累计」
          │    └ QGridLayout 4列 × StatCard×7
          │         累计浏览 | 点赞 | 点踩 | 收藏
          │         累计评论 | 分享 | 附件下载
          └ SectionCard「热门知识 Top 5」
               └ QTableWidget#DataTable (7列)
                    标题|分类|浏览|点赞|收藏|评论|热度
```

**StatCard 用法**：

```cpp
auto *card = new StatCard(QStringLiteral("知识总数"), QString(), QString(), parent);
card->setValue(QLocale().toString(count));
card->setHint(QStringLiteral("可选说明"));  // 一般留空
```

**权限分支**：

```cpp
if (!Session::instance().hasPermission("statistics:view")) {
    showWelcomeOnly();  // 隐藏 statsPanel，改 welcomeHint
    return;
}
```

### 3.2 统计页 statistics（模板 A 增强）

```
QVBoxLayout(0) → PageContainer#PageContent
├ QHBoxLayout
│    ├ #PageTitle「数据统计报表」
│    ├ stretch
│    └ GhostButton「刷新」
├ #LoadingLabel / #ErrorLabel
└ bodyPanel
     ├ SectionCard「知识状态概况」→ StatCard 4×2（同 dashboard 前 8 项）
     ├ SectionCard「全库互动累计」→ StatCard 4+3（同 dashboard 互动 7 项）
     ├ SectionCard「热门知识排行（已上线）」
     │    └ QTabWidget
     │         ├ 浏览量 Top10  → DataTable #,标题,分类,指标,浏览
     │         ├ 点赞 Top10
     │         ├ 收藏 Top10
     │         ├ 评论 Top10
     │         ├ 下载 Top10
     │         └ 综合热度 Top10 → 6列（含热度分、点赞）
     └ SectionCard「搜索词分析」
          └ QTabWidget
               ├ 热门搜索词 Top10
               └ 无结果搜索词 Top10
```

**Tab 内表格高度（关键）**：

```cpp
// 固定行高 42，关闭表格纵向滚动，由 PageContainer 整体滚动
table->setVerticalScrollBarPolicy(Qt::ScrollBarAlwaysOff);
table->setMinimumHeight(headerH + 42 * rowCount + frame*2 + 8);
table->setMaximumHeight(same);
// Tab 切换时 syncTabWidgetHeight(tabs) 防止 pane 高度跳动
```

**并发加载**：statistics 页 8 个接口并行，`pendingLoads` 计数全部完成后隐藏 Loading、显示 bodyPanel。

## 四、API 与数据模型

### 4.1 接口一览

| 方法 | 路径 | 用于 | 权限 |
|---|---|---|---|
| GET | `/statistics/overview` | dashboard + statistics 概况 | `statistics:view` |
| GET | `/statistics/hot-article` | statistics 各 Tab | `statistics:view` |
| GET | `/statistics/hot-keyword` | statistics 搜索词 | `statistics:view` |
| GET | `/statistics/low-quality` | （可选 P1）低质量报表 | `statistics:view` |

### 4.2 overview 响应字段 → StatCard 映射

| 字段 | StatCard 标签 |
|---|---|
| `articleTotal` | 知识总数 |
| `onlineCount` | 已上线 |
| `pendingAuditCount` | 待审核 |
| `draftCount` | 草稿 |
| `todayViewCount` | 今日浏览 |
| `todaySearchCount` | 今日搜索 |
| `offlineCount` | 已下线 |
| `rejectedCount` | 已驳回 |
| `totalViews` | 累计浏览 |
| `totalLikes` | 累计点赞 |
| `totalDislikes` | 累计点踩 |
| `totalFavorites` | 累计收藏 |
| `totalComments` | 累计评论 |
| `totalShares` | 累计分享 |
| `totalDownloads` | 附件下载 |
| `topHotArticles[]` | dashboard Top5 表格 |

### 4.3 hot-article 参数

| 参数 | 说明 |
|---|---|
| `page` | 固定 1 |
| `pageSize` | dashboard 不需要；statistics 用 **10** |
| `sortBy` | `VIEW` / `LIKE` / `FAVORITE` / `COMMENT` / `DOWNLOAD` / `COMPOSITE` |
| `onlyOnline` | `true`（仅已上线） |

### 4.4 hot-keyword 响应结构

```json
{
  "hotKeywords": { "total": 100, "page": 1, "pageSize": 10, "list": [{ "keyword": "退款", "searchCount": 42 }] },
  "zeroResultKeywords": [{ "keyword": "xyz", "searchCount": 3 }]
}
```

Qt 解析见 `HotKeywordReport::fromJson`。

### 4.5 StatisticsService 示例

```cpp
service.fetchOverview([](bool ok, const StatisticsOverview &data, const QString &msg) { ... });
service.fetchHotArticles(QStringLiteral("VIEW"), 10, [](bool ok, const QVector<HotArticleItem> &list, ...) { ... });
service.fetchHotKeywords([](bool ok, const HotKeywordReport &report, ...) { ... });
```

## 五、迁移对照清单（sibling → tuandui）

### 5.1 首页 dashboard

| 检查项 | 应有实现 | 常见错误 |
|---|---|---|
| 滚动容器 | `PageContainer` | 整页 QVBoxLayout 无滚动 |
| 欢迎条 | `#WelcomeBanner` 蓝渐变 | 纯文字 QLabel |
| 用户名 | `Session::user()` 动态问候 | 写死「管理员」 |
| 加载态 | LoadingLabel → 隐藏 → 数据区 | 无反馈 |
| 失败态 | ErrorLabel + Ghost 重试 | QMessageBox 弹窗 |
| 概况卡片 | 8 个 StatCard / 4 列 | 单行 QLabel 数字 |
| 互动卡片 | 7 个 StatCard | 遗漏分享/下载 |
| 热门表 | SectionCard 包裹 DataTable | 裸表格 |
| 无权限 | 仅 WelcomeBanner + hint | 空白页或报错 |

### 5.2 统计 statistics

| 检查项 | 应有实现 | 常见错误 |
|---|---|---|
| 页标题 | PageTitle 17px | 无标题 |
| 刷新 | GhostButton 调 refreshPage | 无刷新 |
| 概况/互动 | 与 dashboard 字段一致 | 两套不同指标 |
| 热门 Tab | 6 个维度 Top10 | 仅 1 个浏览榜 |
| 搜索词 | 热门 + 无结果 两 Tab | 缺失 |
| Tab 高度 | fitTop10Table + syncTabWidgetHeight | Tab 内双滚动条 |
| 并发加载 | pendingLoads=8 | 闪屏/早显 |

## 七、验收清单

### 7.1 首页

- [ ] Win11 深色模式下 StatCard 文字可读

- [ ] 6 个热门 Tab 均可切换且数据正确
- [ ] 表格无内部纵向滚动，整页可滚到底
- [ ] 搜索词两 Tab 有数据
